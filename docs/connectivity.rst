Connectivity module
===================

Connectivity function is as follows:

.. math::
   \tau_i(h) = \textrm{Prob}\{\mathbf{x} \leftrightarrow \mathbf{x}+\mathbf{h} \: | \: s(x) =s_i, \: s(\mathbf{x}+\mathbf{h})=s_i \}
   
where :math:`i \in \{1,2,3\}` is category index
